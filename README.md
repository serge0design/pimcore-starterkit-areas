# Pimcore StarterkitAreasBundle

A collection of some usable Areas.

### Installation

```json
"require": {
  "serge0design/starterkit-areas": "^1.0.0",
}
```

or:

- Execute: $ `composer require serge0design/pimcore-starterkit-areas`

- Execute: $ `bin/console pimcore:bundle:enable StarterkitAreasBundle`
- Execute: $ `bin/console pimcore:bundle:install StarterkitAreasBundle`
